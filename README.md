Interactive web app for planning the IT Cybersecurity degree pathway at Los Angeles City College (LACC). Built with HTML, CSS, and JavaScript — no frameworks, no dependencies.

🌐 **Live demo:** [nattapongsindhu.github.io/IT-Cybersecurity-Planner](https://nattapongsindhu.github.io/IT-Cybersecurity-Planner)

---

## What It Does

- Displays the full LACC IT Cybersecurity course sequence
- Lets users track completed and in-progress courses
- Maps coursework to CompTIA certification targets (A+ → Network+ → Security+ → CySA+ → PenTest+)

---

## Tech Stack

- HTML · CSS · JavaScript
- Zero external dependencies
- Deployed via GitHub Pages

---

## Skills Demonstrated

- DOM manipulation and event handling
- Responsive layout without frameworks
- Translating a real academic program into a functional UI

---

## Background

Built as a practical tool for my own degree planning at LACC. The goal was to have a single view of the entire cybersecurity pathway — courses, sequence, and cert targets — rather than digging through PDF catalogs.
